`title` **(essential)**
- post에 보여질 제목.
- 파일 명을 그대로 쓸려고 했지만 한글과 공백 문자로 인해서 경로 문제가 생길 것 같아서 따로 파일 작성

`date` **(essential)**, `update` (optional)
- sitemap, rss feed에 보여질 날짜.
- date는 작성된 날짜를, update에는 수정된 날짜를 작성할 것.
	- date가 우선적으로 사용되는 데이터이긴 함.
	- 작성 일이 같다면 시간을 다르게 설정해주는 게 좋음.

`tags`
- 보다시피 태그. 여러 개 작성할 수 있음.
- 아직 블로그에는 태그를 통한 게시글 접근 기능을 만들어 두진 않음...
	- (그냥 귀찮았다.)
- 사용한다면 아래 태그를 사용할 예정이긴 하다.
	- `devlog`
	- `study`
	- `project`
	- `pleasant looking trash`
