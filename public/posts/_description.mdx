잡동사니가 모이는 공간.

`_description.mdx` 이 파일은 폴더 설명 및 메타데이터 작성을 위해 만듦.

---

`title`
- 브라우저나 OpenGraph에 보여질 제목을 적는다.
- 되도록이면 짧은 제목을 적을 것.

`description`
- 블로그에서 보여질 폴더 제목을 적는다.
- 길어도 상관없긴 하지만 `title` 속성과 마찬가지로 짧을수록 좋음.

`order`
- 게시글 정렬 기능.
- true 값을 가지면 최신 순으로,
- false 값을 가지면 오래된 순으로 정렬.
	- 목차 순서대로 게시글을 보여줄려면 false 로 설정하는게 좋음.

---

이 파일을 작성하지 않으면 해당 폴더 명으로 `title`과 `description`이 정해짐.

함수에 `_`로 시작하는 mdx 파일이나 `_description.mdx` 파일은 post로 읽어들이지 않도록 코드를 작성해놓았음.
- 이 파일로 인해서 새로운 게시글이 생성되거나 그러진 않음.
- mdx 파일명 앞에 `_`만 붙여서 메모용으로 사용해도 상관없음.
- 초안을 적어두고 보관할 공간 `draft` 폴더도 만들어두었으니 여길 사용해도 상관없음.
	- 폴더와 관련이 없거나 따로 파일을 관리하고 싶다면 `draft` 폴더를 사용할 것.
	- (obsidian을 통해서 게시글을 작성 중이기에..)
